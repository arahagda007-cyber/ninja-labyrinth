# ?? Ninja of the Lost Labyrinth

Un juego de laberinto desarrollado con Pygame. Controla a un ninja que debe encontrar la salida en un laberinto lleno de obstáculos.

## ?? Características

- Laberinto interactivo con paredes verdes
- Objetivo rojo que indica la salida
- Controles simples con teclado
- Sistema de reinicio integrado
- Mensaje de victoria al completar el nivel

## ??? Controles

| Tecla | Acción |
|-------|--------|
| ? ? ? ? | Mover al ninja |
| R | Reiniciar el juego |
